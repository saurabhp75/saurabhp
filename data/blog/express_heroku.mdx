1. Heroku login.
2. Git status → git init.
3. Git add filename.
4. Git commit -m “commmit message”
5. Heroku login.
6. Heroku create.
7. Git remote -v.
8. git push heroku master.
9. Heroku logs (shows logs for app in current dir)
10. we can run any command (ls, npm etc) using.

```shell
$Heroku run bash-command
```
